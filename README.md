# Bacon Ipsum Fetch (Async/Await)

A simple and modern web app that fetches and displays Bacon Ipsum text using the **Fetch API** with **async/await**.

---

## How to Run

1. Download or clone this folder.
2. Open `index.html` in your browser.
3. The app will automatically fetch and display Bacon Ipsum data.
4. Click **"Fetch Bacon Text"** to refresh with new content.

---

## Features

- Built with modern **async/await** syntax
- Proper **try...catch** error handling
- Styled completely with **Tailwind CSS**
- Responsive and minimal UI design
- Includes a **loading indicator** and one-click refresh

---

## API Used

**Bacon Ipsum API**  
**Endpoint:**  
`https://baconipsum.com/api/?type=all`

---

## Bonus Features

- Animated loading indicator
- Refresh button for new data
- Fully responsive layout
